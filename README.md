# closedverse-video-support
so its closedverse, but with video mp4 support and all the features from the other closedverse clones merged into one (thanks arian)

# Requirements
  * Python 3.6.2?
  * Django
  * urllib3?
  * lxml
  * passlib
  * bcrypt
  * mysqlclient if you're using mysql
  * pillow
  * imghdr?
  * markdown
